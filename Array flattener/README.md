## Array flattener


**Level:** Medium-High

**This exercise will assess the candidate's knowledge in:**
- Functions
- Arrays
- Problem solving
- Recursive functions

**Problem:**
```javascript
/**
 Create a function called `flatten` that will receive an array and will flatten its content.
 */

const test1 = [1, [2, [3], 4], [5]]
flatten(test1) // [1,2,3,4,5]

const test2 = [[[3]]]
flatten(test2) // [3]
```
